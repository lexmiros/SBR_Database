About

Project

This project is an extension of a prescribed university assignment. The scope of the assignment was to create a database design from scartch, chosing any universe of discourse. The universe chosen for the project was based on a real cattle company, Seifert Belmont Reds, at a reduced scale for feasbility. Further discussion on the project can be found in the report. For the database design, the project required:

  • At least 5 entities
  • At least 5 relationships

Specifications

The project criteria required MySQL to be used as the DBMS, and that the created database structure be normalised to BCNF or 3NF. The ER diagram, Schema, and BCNF normalisation used for this project can be found in the report. For implementation, the project required:

  • A join query
  • Aggregation query
  • Aggregation query with group-by
  • Delete with cascade
  • Update

The above requirements had to be implemeneted using basic SQL.

Extension

Two main extensions were added to the project for personal learnings. The project was implemented with a complete GUI using Flask, HTML and CSS, and a linear regression model to predict cattle prices was added.

The linear regression model was built from real cattle sales data from Seifert Belmont Reds. The data used to build the model has not been included at the request of the owners, but an analysis of the models performance can be found in the reports tab.
